# Implementation-of-Lexical-Analysis
## Description
For this evidence, the language chosen was all the possible combinations of ab1.<br>
The rules were the following:<br>
- Must have ab1 or 1ba
- Must end with ba.
The **modeling tecnique** used was a Deterministic Finite Automata (DFA) to represents my solutions. DFA was develop because of the predictable and deterministic behavior in its definition. By the rules required in the lexical analysis, the DFA was implemented to assure that the AUTOMATA fulfilled the necessary processes.
